# Release guide

## SemVer recommandé
- **v0.1.0** : MVP initial
- **v0.1.1** : correctifs (bugfix)
- **v0.2.0** : maintenabilité et/ou nouvelles fonctionnalités

## Checklist release
- Build OK
- Tests OK
- Seed/Reset OK (si disponibles)
- Docs à jour et cohérentes

## Tag Git
```bash
git tag v0.1.0
git push --tags
```
